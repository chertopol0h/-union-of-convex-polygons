# Объединение выпуклых полигонов

Этот проект содержит реализацию алгоритма объединения выпуклых полигонов на Python с использованием Tkinter для визуализации.

## Описание файлов

- `task11.py` - основная реализация алгоритма объединения выпуклых полигонов
- `task11_incorrect.py` - альтернативная реализация (возможно, содержащая ошибки)

## Использование

1. Запустите программу:
```bash
python task11.py
